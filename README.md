# `payserver`

**A easy way to get paid. 使用个人支付，让收款更简单。**

*:warning: 由于目前个人收款的政策调整，本项目暂时停止开发，待政策明确后继续。---2021.10*
  
</div>

---

## 特点

- **实时通知**: 设备收到付款时立即通知后端服务
- **多平台支持**: 支持Android和iOS(部分)
- **自动和手动模式**: 支持手动模式，不错过任何一笔收款
- **简单的API**: 轻松地集成到你的服务中
- **SaaS和独立部署**: 提供SaaS服务，亦可独立部署
